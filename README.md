# 💸 Harcama Takip Uygulaması

> Mobil Programlama | Ionic Framework ile geliştirilmiş günlük harcama takip uygulaması.

---

## 📱 Özellikler

- ✅ Harcama ekleme (ad, tutar, kategori)
- ✅ Kategoriye göre renkli gösterim
- ✅ Toplam harcama hesaplama
- ✅ Kaydırarak silme (ion-item-sliding)
- ✅ Silme öncesi onay mesajı
- ✅ Toast bildirimleri
- ✅ LocalStorage ile kalıcı veri saklama
- ✅ Harcama tarihi ekleme

---

## 🛠️ Kullanılan Teknolojiler

| Teknoloji | Versiyon |
|-----------|----------|
| Ionic Framework | 7+ |
| Angular | 17+ |
| TypeScript | 5+ |
| Capacitor | 5+ |

---

## 🚀 Kurulum

### Gereksinimler

- Node.js (v18+)
- npm
- Ionic CLI

### Adımlar

```bash
# Repoyu klonla
git clone https://github.com/KULLANICI_ADIN/harcama-takip.git

# Proje klasörüne gir
cd harcama-takip

# Bağımlılıkları yükle
npm install

# Uygulamayı çalıştır
ionic serve
```
